- 说明：计算target里对象的权重
- 例子：
  - target_rst源数据：target_rst[11] = { [8,10,11]}
  - S_Q源数据：
    - 一、计算target_rst的权值：
      - 从B_S拿下在8的位置（8: ['11', '169']），拿到11位置为0--->拿target对象位置index=0
      - 从S_Q拿下8的权值数据 8: [0.55, 0.59],再根据11的index位置拿到对应权值0.55(index=0)

      - 计算target权重组合：target_qz[11] = sum([0.55,0.66,0.77])，其他以此类推
        - target_rst[11] = [0,1,2,3], target_qz[11] = [0,1,2,3]
    - 二、计算target_dna的权值：
